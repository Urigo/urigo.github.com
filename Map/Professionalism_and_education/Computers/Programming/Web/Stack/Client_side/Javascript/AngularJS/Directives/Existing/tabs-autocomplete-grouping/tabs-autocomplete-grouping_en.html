We want one directive to rule them all.  It is an input field that:

  1. has autocomplete
  2. has tags
  3. has grouping and hierarchy
  4. can get objects as autocomplete results

There are a lot of solutions out there, but non of them is good enough:

UI-Bootstrap's typeahead

Has great autocomplete with objects and promise support. but it doesn't supprt tags or grouping, issue

Select2 Angular-port

Has autocomplete, tags, grouping but uses JQuery with horrible syntax!

ngInputTags

Great support for tags tags, autocomplete, but does not support objects, just strings and does not support grouping

angular-tags

Support tags and uses ui-bootstrap's typeahead with it's limitations
