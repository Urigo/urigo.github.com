- תתחילו עם ה Tutorial המעולה. הוא לוקח מעט זמן יחסית, הוא מעולה, מכסה הרבה נושאים עיקריים ונותן את התחושה הכי טובה של מה זה AngularJS, יותר טוב מכל הסבר של אנשים.
- אחרי שסיימתם את ה Tutorial והחלטתם שכדאי להמשיך, תעברו ל Developers guide שמסביר יותר לעומק נושאים חשובים. תמיד תשמרו לינק ל API reference שתמיד כדאי שיהיה זמין.

דוגמאות לאפליקציות שנכתבו ב AngularJS

tutorial חדש ומעודכן אבל עם פחות פירוט. כדאי לעשות אותו אחרי שעושים את הראשון.
